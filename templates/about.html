À propos de l'Analyse des Transactions BIAT

Cette application a été développée pour analyser les transactions bancaires de la BIAT Banque Internationale Arabe de Tunisie. Elle permet de visualiser les données de transaction, d'identifier des tendances et de fournir des insights utiles pour la prise de décision.

Fonctionnalités

  • Visualisation des transactions par type, heure, et autres dimensions.
  • Calcul des statistiques clés telles que le montant total, le montant moyen, et le nombre total de transactions.
  • Interface utilisateur intuitive avec des graphiques interactifs.
  • Possibilité d'exporter les données analysées.

Technologies Utilisées

  • Flask pour le backend web.
  • Pandas pour la manipulation des données.
  • Matplotlib et Seaborn pour la visualisation des données.
  • Bootstrap pour le design de l'interface utilisateur.
  • Chart.js pour les graphiques interactifs.
